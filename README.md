## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ajmalaju06/JSCodeRiddles.git

   ```

2. Redirect to the project

   ```bash
   cd JSCodeRiddles

   ```

3. Run the project.

   ```bash
   npm run dev
   ```
